## Assessment Overview

The assessment is divided into three parts, each addressing a distinct area of QA automation:

- **Part 1: Debugging Flaky Playwright Tests** — Identifying and fixing reliability issues in an existing Playwright test suite for a login flow.
- **Part 2: Test Automation Framework Design** — Designing a scalable, maintainable automation framework structure for a multi-tenant SaaS platform.
- **Part 3: API + UI Integration Testing** — Building an end-to-end test that validates project creation across API, web UI, and mobile layers.
